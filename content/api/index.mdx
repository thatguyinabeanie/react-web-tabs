# API Reference

Complete API documentation for all React Web Tabs components.

## Core Components

- **Tab** - Individual tab button with WAI-ARIA attributes
- **TabList** - Container for tab buttons
- **TabPanel** - Content container for a tab
- **Tabs** - All-in-one container (uses TabProvider)
- **TabProvider** - Context provider for tab state management

All components follow the [WAI-ARIA Authoring Practices 1.1](https://www.w3.org/TR/wai-aria-practices-1.1/#tabpanel) for accessibility.
