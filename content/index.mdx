# React Web Tabs

Modular and accessible React tabs according to the [WAI-ARIA Authoring Practices 1.1](https://www.w3.org/TR/wai-aria-practices-1.1/#tabpanel).

## Installation

This package depends on React ^16.3.0

Using [npm](https://www.npmjs.com/):
```bash
npm install --save @thatguyinabeanie/react-web-tabs
```

Using [yarn](https://yarnpkg.com/en/):
```bash
yarn add @thatguyinabeanie/react-web-tabs
```

Then with a module bundler like [webpack](https://webpack.js.org/) you can import it like usual:

```js
// using ES6 modules
import { Tabs, Tab, TabPanel, TabList } from '@thatguyinabeanie/react-web-tabs';

// using ES6 Partial imports
import Tabs from '@thatguyinabeanie/react-web-tabs/lib/Tabs';
import Tab from '@thatguyinabeanie/react-web-tabs/lib/Tab';
import TabPanel from '@thatguyinabeanie/react-web-tabs/lib/TabPanel';
import TabList from '@thatguyinabeanie/react-web-tabs/lib/TabList';

// using CommonJS modules
var Tabs = require('@thatguyinabeanie/react-web-tabs').Tabs;
var Tab = require('@thatguyinabeanie/react-web-tabs').Tab;
var TabPanel = require('@thatguyinabeanie/react-web-tabs').TabPanel;
var TabList = require('@thatguyinabeanie/react-web-tabs').TabList;
```

The UMD build is also available on unpkg:

```html
<script src="https://unpkg.com/@thatguyinabeanie/react-web-tabs/dist/react-web-tabs.min.js"></script>
```

## Basic Usage

```jsx
import React, { Component } from 'react';
import { render } from 'react-dom';
import { Tabs, Tab, TabPanel, TabList } from '@thatguyinabeanie/react-web-tabs';

class App extends Component {
  render() {
    return (
      <Tabs
        defaultTab="one"
        onChange={(tabId) => { console.log(tabId) }}
      >
        <TabList>
          <Tab tabFor="one">Tab 1</Tab>
          <Tab tabFor="two">Tab 2</Tab>
          <Tab tabFor="three">Tab 3</Tab>
        </TabList>
        <TabPanel tabId="one">
          <p>Tab 1 content</p>
        </TabPanel>
        <TabPanel tabId="two">
          <p>Tab 2 content</p>
        </TabPanel>
        <TabPanel tabId="three">
          <p>Tab 3 content</p>
        </TabPanel>
      </Tabs>
    );
  }
}

render(<App/>, document.getElementById('app'));
```

## Advanced Usage with TabProvider

If you need to mix in other elements between tabs:

```jsx
import React, { Component } from 'react';
import { render } from 'react-dom';
import { TabProvider, Tab, TabPanel, TabList } from '@thatguyinabeanie/react-web-tabs';

class App extends Component {
  render() {
    return (
      <TabProvider defaultTab="one">
        <section className="my-tabs">
          <TabList className="my-tablist">
            <Tab tabFor="one">Tab 1</Tab>
            <span className="divider">•</span>
            <Tab tabFor="two">Tab 2</Tab>
            <span className="divider">•</span>
            <Tab tabFor="three" className="my-tab">Tab 3</Tab>
          </TabList>
          <div className="wrapper">
            <TabPanel tabId="one">
              <p>Tab 1 content</p>
            </TabPanel>
            <TabPanel tabId="two">
              <p>Tab 2 content</p>
            </TabPanel>
            <TabPanel tabId="three">
              <p>Tab 3 content</p>
            </TabPanel>
          </div>
        </section>
      </TabProvider>
    );
  }
}

render(<App/>, document.getElementById('app'));
```

Every component supports adding additional props like custom className's or data attributes.

## Styles

Some basic styles are provided but they are optional as the tabs are fully functional without styling. Both minified and unminified versions are available in the `/dist` folder.

With webpack:
```js
import '@thatguyinabeanie/react-web-tabs/dist/react-web-tabs.css';
```

## Keyboard Support

The following keys can be used to navigate between tabs when in focus, according to the [WAI-ARIA Authoring Practices 1.1](https://www.w3.org/TR/wai-aria-practices-1.1/#tabpanel).

Horizontal tabs:
- `←` Navigate to previous tab
- `→` Navigate to next tab
- `HOME` Navigate to first tab
- `END` Navigate to last tab

Vertical tabs:
- `↑` Navigate to previous tab
- `↓` Navigate to next tab
- `HOME` Navigate to first tab
- `END` Navigate to last tab

According to the WAI-ARIA Practices 1.1 only the active tab should receive focus upon entering and leaving the tab list. To make all tabs focusable you can override this behavior by adding the `focusable` flag to each `<Tab>` component:

```jsx
<Tab focusable tabFor="my-tab">React web tabs</Tab>
```
